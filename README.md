- `solution.py` — main solution script
- `puzzle_input.txt` — input data file

## How to run

Make sure `puzzle_input.txt` is in the same directory as the script, then:

```bash
python solution.py
```

Expected output:

```

Part 2: <answer>
```

## Requirements

Python 3.14+, no external dependencies.
